ubu_sys_config
=========

Роль для конфигруции ВМ после создания.

<br>

---

Требования
------------

ОС: Ubuntu (проверлось на версиях 20.04 и 22.04) 
Выполняется от пользователя с повышенными привелегиями(sudo)

<br>

---

Пример применения роли в плейбуке
----------------

```
- hosts: localhost 
  remote_user: root 
  roles: 
    - ubu_sys_config
```
